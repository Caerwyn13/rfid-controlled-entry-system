Users Table: Stores user data
```
CREATE TABLE Users (
    RFID_key VARCHAR(20) PRIMARY KEY,
    fname VARCHAR(100),
    lname VARCHAR(100),
    email VARCHAR(100),
    department_id INT,
    last_updated TIMESTAMP,
    FOREIGN KEY (department_id) REFERENCES Departments(department_id)
);
```

Departments Table: Stores department names
```
CREATE TABLE Departments (
    department_id INT PRIMARY KEY,
    department_name VARCHAR(100)
);
```

Access Log Table: Tracks each access event
```
CREATE TABLE AccessLog (
    log_id INT PRIMARY KEY,
    RFID_key VARCHAR(20),
    accessed_timestamp TIMESTAMP,
    received_timestamp TIMESTAMP,
    action_type CHAR(10),
    is_authorized BOOLEAN,
    FOREIGN KEY (RFID_key) REFERENCES Users(RFID_key)
);

```
